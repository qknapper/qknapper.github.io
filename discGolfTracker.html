Disc Golf Tracker -

    > Back to Portfolio

Disc Golf Tracker is a java project designed to record and maintain a text file log of disc golf scores and data based on user input. Through a prompt based interface in the user's console, the Disc Golf Tracker first takes in the name, date, and number of holes in a course, then prompts the user for the par and their performance on each hole, calculating the cumulative, relative relationship between the two. After completing the data collection, the Disc Golf Tracker then exports the data to a new text file or updates a previously identified log.
